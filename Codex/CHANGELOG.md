# Changelog

## 1.0.0

- Первая версия MovieTracker CLI Codex.
- TMDB из исходного ТЗ заменён на Kinopoisk API Unofficial.
- Добавлены команды поиска, деталей, watchlist, оценок, рекомендаций, popular/trending, auth token, cache и init.
- Добавлены инструкции для dev-инициализации, тестирования и пользовательской установки.
